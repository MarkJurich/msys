Reporting Bugs

So you think you found a bug and want to report the problem? When reporting your bug please be kind and provide as much detail as possible about how you experience your bug. Give a small test case that demonstrates the bug if it concerns GCC. Give MSDN documentation references if the bug involves the lacking features in the w32api or mingw-runtime. SourceForge gives us nice tools to keep track of your bug reports, so please follow this link to file your report. You will need a SourceForge login account. If you don't have one then follow this link to register for one.
