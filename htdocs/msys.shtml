MSYS

MSYS was created out of a long lived desire to provide the MinGW community a Minimal SYStem with which a configure script could be executed. We finally have that available and is downloadable from the SourceForge file distribution system which MinGW is using.

The current version is msys-1.0.2 and you need to download either the i686 binary distribution or the i386 binary distribution. The source distribution is delivered in this package and includes the source for both the runtime and all the packages from which the hand selected binaries were taken.

To install chose a drive device and unpack the binary distribution in the root of that drive. The read the documentation in \msys\1.0\usr\doc\msys, especially the README file.

There is also a new mail list to discuss the use of MSYS called mingw-msys@lists.sf.net. See the Mailing Lists page for a more information on this list.
